# 饱了吧外卖订餐系统

- 站在系统设计者的角度从零开始构建一个外卖订餐系统

## 技术栈

### 后端

#### 基础设施

- 容器管理：docker
- 数据库：mysql
- 注册中心/配置中心：nacos
- 消息队列：rabbitmq
- 文件服务器：fastdfs

#### 框架

- 微服务框架：spring cloud
- 单元测试
  - junit5
  - mockito
  - spring boot test
  - h2
- 文档管理：swagger
- 服务调用：open feign
- 服务熔断/降级：alibaba sentinel

#### 运维

- 持续集成：github action
